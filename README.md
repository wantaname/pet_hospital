## 前言

如果你通过“虚拟宠物医院”搜索到了这个项目，那么你很有可能是华师大的，并且说明《开发实践》这门课已经有五年以上没有换过题目了。

个人觉得我校软件专业不应该把重心放在做这种无意义的课程项目上，纯粹浪费同学们的时间。

## 技术栈

前端：Vue Vue-router Element-ui Three.js 

后端：Python Flask

数据库：Mysql
